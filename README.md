### 📋 重构说明

本项目已从Vue.js应用重构为单页HTML网站，原因如下：
- **简化维护**: 移除复杂的前端框架依赖
- **提升性能**: 纯HTML/CSS/JS，加载速度更快
- **降低成本**: 无需构建过程，可直接部署到任何静态服务器
- **移除API依赖**: 不再依赖壮壮API的官网相关接口
- **管理端解耦**: 御御管理页面已隐藏官网管理菜单

## 🛠️ 开发指南

### 本地开发

```bash
# 启动本地服务器
npm run dev
# 或
python -m http.server 8080

# 访问地址
http://localhost:8080
```

### 部署方式

#### 1. 静态服务器部署
直接将整个目录上传到任何静态服务器即可。

#### 2. CDN部署
- 上传到CDN服务商
- 配置域名解析
- 启用GZIP压缩

#### 3. GitHub Pages
```bash
# 推送到GitHub
git push origin main

# 在GitHub仓库设置中启用Pages
# 选择main分支作为源
```

#### 4. Vercel/Netlify
- 连接GitHub仓库
- 自动部署，无需构建配置
